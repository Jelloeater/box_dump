# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please open an issue on GitHub. We appreciate responsible disclosure.

## Security Considerations

- This tool runs system commands to list installed packages
- JSON and SQLite files are stored locally
- GitHub credentials should be handled securely
- Review before running on production systems
